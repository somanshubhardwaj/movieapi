API Documentation

GET /api/data

Returns data from the JSONPlaceholder API.

GET /api/movie/:id/similar

Returns similar movies for a given movie ID.

GET /api/tv/:id/similar

Returns similar TV shows for a given TV show ID.

GET /api/person/:id/combined_credits

Returns combined credits for a given person ID.

GET /api/movie/:id/reviews

Returns reviews for a given movie ID.

GET /api/tv/:id/reviews
